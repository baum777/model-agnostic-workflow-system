# mcp-server-creation

Reusable repo-local skill for designing and scaffolding remote MCP servers with explicit tool contracts, safe auth choices, and honest operational defaults.

## Use This Skill For

- internal remote MCP tool servers
- ChatGPT App backends
- API integration servers
- first-pass TypeScript/Node scaffolds with docs, examples, and review criteria

## Included Files

- `SKILL.md`
- `templates/`
- `examples/`
- `checklists/`

## Minimal Invocation

`Use the repo-local skill mcp-server-creation at skills/mcp-server-creation to design and scaffold a remote MCP server that exposes read-only tools for searching internal docs. Use TypeScript, noauth on a private network, explicit schemas, structured logging, and deployment notes.`

## Output Expectation

The skill should produce:

- an architecture brief
- a tool catalog
- schemas and result contracts
- a server scaffold
- environment notes
- a test and deployment checklist
